
### 간단한 소개 🤚
6년간 웹 서비스 개발 경험을 기반으로 React 중심의 UI·상태 관리 역량을 확장 중인 전환형 프론트엔드 개발자입니다.
태양광 모니터링 Hybrid/Web 프로젝트에서 실시간 데이터 UI, WebView 성능 개선, 사용자 흐름 설계를 수행하며
UI 구조와 상호작용에 대한 흥미를 React로 확장하게 되었습니다.

현재는 개인 프로젝트를 통해
* 컴포넌트 구조화
* 커스텀 훅 기반 UI 모듈 설계
* Redux 상태 관리
* 무한 스크롤 및 반응형 UX 개선
등 프론트엔드 실전 패턴을 구현하며 성장하고 있습니다.
단순 기능 구현이 아닌, 사용 경험과 데이터 흐름을 연결하는 UI 개발자로 나아가고 있습니다.
